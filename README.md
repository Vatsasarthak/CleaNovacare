# 🚀 CleaNovacare

> ⚠️ This project is currently under active development .

Cleanova DryClean is a modern laundry and dry cleaning platform built for seamless doorstep laundry services. The platform provides a premium customer experience with a clean modern UI, real-time order management, OTP authentication, and powerful admin controls.

---

## ✨ Features

### 👤 Client Dashboard
- 📦 Book laundry services online
- 🚚 Schedule pickup & delivery
- 📍 Add/manage addresses
- 📱 OTP-based authentication
- 📊 Real-time order tracking
- 💳 View receipts & order history
- 🔍 Search & filter services
- 🧺 Dynamic service selection

### 🛠️ Admin Dashboard
- 📈 Dashboard analytics
- 👥 Customer management
- 📦 Order management
- 🧾 Receipt generation
- 💰 Pricing management
- 🔄 Update order statuses
- 📊 Reports & statistics
- 🧺 Service management
